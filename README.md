# 青舟 (qzboat)

Professional AI SaaS Platform – Next.js boilerplate for building profitable SaaS with auth, payments, i18n, newsletter, dashboard, blog, docs, and more.

## Development

```bash
npm install
npm run dev
# open http://localhost:3000
```


## Environment

Copy `env.example` to `.env.local` (or set in Vercel):
- `NEXT_PUBLIC_GOOGLE_ANALYTICS_ID` - Google Analytics ID
- `NEXT_PUBLIC_SITE_URL` - Site URL (e.g., https://www.qzboat.com)
- Database and other service configurations as needed

## Deploy

- Push to GitHub and connect to Vercel.
- Configure your environment variables in Vercel, redeploy.
